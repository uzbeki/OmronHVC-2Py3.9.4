Now this repo does support python3.9.4
make sure you have Camera sensor plugged into your pc and its driver installed

New features:
    - it logs detected inputs to the Logs.log file
    - it supports pythons latest version

1. clone this repository
2. make sure you have python 3.9.4 (win32) installed. win64 is not supported due to .dll file incompatability
3. pip install -r requirements.txt
4. run py -3 .\execution.py COM3 9600 ON
